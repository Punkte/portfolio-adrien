# How to install

#### Requirements
* [Docker](https://docs.docker.com/)
* [Docker-compose](https://docs.docker.com/compose/)

#### Launch the project

Create a `.env` file and copy the content of the [.env.example](.env.example) file.  

Build images in docker-compose  
```bash
$ docker-compose build
```
Launch containers  
```bash
$ docker-compose up # -d
```
